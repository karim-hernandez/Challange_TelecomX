## Descripción
Este proyecto se enfoca en el análisis de la evasión de clientes ( o Abandono) en una empresa de telecomunicaciones, como parte del Challenge de aLugara G9 "Telecom X". El objetivo principal es comprender a fondo los factores que contribuyen a que los clientes abandonen el servicio, con el fin de proponer estrategias para la retención de los clinetes. A través del análisis exploratorio de datos, limpieza y procesamiento, se identificaron patrones y se ofrecen algunas recomendaciones para reducir el abandono.

## Instalación y ejecución
1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/karim-hernandez/Challange_TelecomX.git
   ```
2. **Instalar dependencias**:
   ```bash
   pip install pandas numpy matplotlib seaborn plotly ipython
   ```
3. **Abrir el Notebook**:
   ```bash
   jupyter notebook TelecomX_LATAM.ipynb.ipynb
   ```
4. **Ejecutar las celdas**:
   Ejecuta todas las celdas en orden para reproducir el análisis completo.

## Posibles Problemas y Soluciones

*   **Errores de conexión a la URL de datos:** Asegúrate de tener una conexión a internet estable. Si el enlace `datos_url` cambia o deja de estar disponible, el notebook no podrá cargar los datos.

*   **Problemas de versiones de librerías:** Si experimentas errores relacionados con librerías, puede ser derivado de una incompatibilidad de versiones. 

*   **Rendimiento lento en visualizaciones:** Algunas visualizaciones interactivas pueden ser lentas, ten paciencia.
